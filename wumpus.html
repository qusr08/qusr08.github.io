Game Description

Wumpus is a game about navigating your way around an unknown dungeon in search of a creature called the Wumpus. You must find a weapon like a sword, bow, or trap in order to kill the Wumpus where it stands. There are torches and flashlights around the map too to help you discover your way, but there are also various pits that can cut your adventure short. You can only see what is on the tiles directly around you, with all other tiles shrouded by darkness. The Wumpus is lurking and moves around the map to try and sneak attack you. The goal of the game is to kill the Wumpus!

This project was created for a computer science class offered at Mount Olive High School. The prompt for this project was only to make a console-based program in Java that had the basic mechanic of trying to search for and kill the Wumpus. I wanted to make it more challenging for myself and really put my programming skills to the test with this project. I not only made it graphical but also added more objects and items onto the map making the game more interesting. I was the only person in my class to do this and my teacher decided to keep the program to show future students as an example and to help explain the project.

My Thoughts

I am very happy with the final product and think it looks polished and presentable. The pixel art I did for this game is definitely some of the best I have been able to achieve, and I believe it pulls the game together nicely. I will not be developing this game further because it was made from scratch in Java and it was for a class. The project as it stands is a good display of my early game development mindset, and I want to keep it as a past game that I can look back on.
